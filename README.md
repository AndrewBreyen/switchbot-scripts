# switchbot-scripts
 
## Setup
1. Clone the repository, and create a `.env` file in the directory. 

## Requirements
pip install python-dotenv

## Running

```
cd path/to/switchbot-scripts

python [SCRIPT]
```

where `[SCRIPT]` is one of the following: 

- switchbot-bot-device-list.py
- switchbot-bot-on.py
- switchbot-bot-off.py
